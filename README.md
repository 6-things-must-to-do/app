# STMT (Six Things Must Todo)

STMT is cross platform application which is made of [React Native](https://reactnative.dev/)

## Skill

- [x] Typescript
- [x] Redux
- [x] Redux Saga
- [x] React Navigation
- [x] Styled Components
- [ ] Fastlane
- [ ] Sentry

## Setup

Need react native development environment prerequisite

&&

```
git clone app ./app;
cd app && yarn;

# IOS
cd ios && pod install
```

```
yarn android

or

yarn ios
```
